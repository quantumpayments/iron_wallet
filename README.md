# iron_wallet

create an ironwallet

see : https://github.com/melvincarvalho/wallet/wiki/Iron-Wallets

# create.sh

    create.sh <database> <wallet>

creates a wallet with

* coinbase : 1000000

# workbot.sh

    workbot.sh <database> <wallet> <workbot>

creates a wallet with

* coinbase : 9000000
* workbot  : 100000
